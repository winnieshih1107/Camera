# 📸 Pose Studio AI — 網美姿勢引導相機

匯入一張喜歡的參考照，AI 自動把照片裡的人物轉成火柴人骨架疊在相機畫面上；
被拍的人走進畫面、姿勢對齊骨架變綠，就是快門時機——還能自動幫你按。

## ✨ 功能特色

### 🤖 AI 姿勢分析與骨架提取
從相簿匯入任何照片，內建 MediaPipe Pose 模型自動偵測照片中的人物，
轉成半透明的火柴人點線圖（綠線＋紫色關節點風格），疊在相機即時畫面上。

### 🎯 AI 即時對齊偵測與自動快門
即時比對鏡頭前的人與參考骨架的重合度（以髖部置中、軀幹長度歸一化後比對 12 個關節，
含鏡像比對）。重合度即時顯示在畫面上，達 **85%** 時骨架由粉轉綠並顯示「Pose Matched!」；
開啟自動快門後，對齊達標會延遲 0.5 秒自動拍攝。

### ✋ 骨架／照片自由位移縮放（多點觸控）
骨架與參考照片可分別用手勢調整：單指拖曳移動、兩指縮放與旋轉；
「鎖定」模式時兩指恢復為相機變焦。電腦上以滑鼠拖曳＋Shift 滾輪操作。

### 📐 九宮格與環境水平線
經典 3×3 三分構圖線（預設開啟），並在匯入照片時自動偵測海平線、桌沿等
關鍵水平邊緣，以金色虛線標示，協助對齊環境、避免畫面歪斜。

### 🎛 收折式控制面板
透明度、顯示模式、手勢對象、AI 開關集中在一塊可收折面板，
一鍵收起、閒置自動收合，把構圖空間還給畫面。

### 📷 完整相機控制
前後鏡頭切換、0.5×–5× 變焦（優先使用硬體變焦）、點擊對焦、手電筒、
曝光／ISO／白平衡／色溫等進階參數（依裝置支援程度顯示）。

### 🧍 內建姿勢庫
咖啡廳、自然山海、街市地標、健身運動四大類引導姿勢，
每個姿勢附擺法要領、構圖輔助線與真實範例照搜尋連結。

## 🚀 使用流程

1. 開啟網址、允許相機權限
2. 點「看範例照」找一張喜歡的姿勢照，或直接從相簿匯入
3. AI 自動抽出骨架疊在畫面上，手動微調位置與大小
4. 開啟「AI 對齊」，請被拍的人對齊骨架——變綠就拍（或交給自動快門）

## 🛠 技術架構

- **單一 HTML 檔**，無建置流程、無框架，純 Vanilla JS
- **AI 模型**：[MediaPipe Pose Landmarker](https://developers.google.com/mediapipe)（tasks-vision，CDN 載入，瀏覽器端推論，照片不上傳）
- **相機**：getUserMedia ＋ ImageCapture 進階控制
- **PWA**：manifest ＋ service worker（離線快取、可安裝）

## 💻 本機開發

npx http-server -p 8787

> AI 模型自 CDN 載入，首次使用需要網路。

## 🔒 隱私

所有影像處理（姿勢偵測、對齊比對）皆在瀏覽器本地執行，
照片與相機畫面不會上傳到任何伺服器。
